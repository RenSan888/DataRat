# :rat: Data Rat

Data Rat is a browser extension and dashboard that helps users understand what they’re agreeing to when they accept cookies or terms of service.

The extension:

- :mag: Detects consent popups and policies

:robot: Uses AI to translate them into simple terms:  

- What data is collected
- How it’s used
- Potential risks

:bar_chart: Gives a quick consent risk score

Instead of blindly clicking “Accept,” users can make informed decisions about their data.

**Tools Used:**

JavaScript, HTML/CSS, Chrome Extensions (Manifest V3), AI API (Gemini)
